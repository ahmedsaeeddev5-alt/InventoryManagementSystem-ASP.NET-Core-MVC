## 🚀 Overview

The **Inventory Management System** is a full-featured web application designed to efficiently manage products and categories in a structured and scalable way.

The project demonstrates **real-world CRUD operations**, **clean architecture implementation**, and **enterprise-level design patterns** using ASP.NET Core MVC.

---

## ✨ Key Features

- 📦 Product Management (Create, Read, Update, Delete)
- 🗂 Category Management with relational structure
- 🔍 Search functionality for fast data filtering
- 📄 Pagination for improved performance and UX
- 🖼 Product image upload and management
- ✅ Server-side validation using Data Annotations
- 🎨 Responsive UI using Bootstrap and Razor Views
- 🧱 Clean separation of concerns using layered architecture
- ⚡ Efficient data handling using Entity Framework Core
- 🧼 Maintainable and scalable code structure

---

## 🛠 Tech Stack

**Backend**

## 🧱 Architecture & Design Principles

This project follows modern software engineering principles:

- 🏛 Repository Pattern (Data access abstraction)
- 🔄 Unit of Work Pattern (Transaction management)
- 🎯 SOLID Principles
- 🔌 Dependency Injection (Built-in ASP.NET Core DI)
- 🧼 Clean and maintainable layered architecture
- 📦 Separation of Concerns (SoC)

## 🎯 What This Project Demonstrates

- Building real-world MVC applications using ASP.NET Core
- Designing scalable and maintainable backend architecture
- Handling relational data (Products ↔ Categories)

- Structuring clean and testable codebase

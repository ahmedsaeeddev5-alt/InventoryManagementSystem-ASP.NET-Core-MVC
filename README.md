# Inventory Management System

A scalable web-based Inventory Management System built using ASP.NET Core MVC, Entity Framework Core, and SQL Server.

## Features

- Product Management
- Category Management
- CRUD Operations
- Repository Pattern
- Unit of Work Pattern
- Dependency Injection
- Product Image Upload
- Search Functionality
- Pagination
- Server-Side Validation
- Responsive UI with Bootstrap

## Technologies

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- LINQ
- Razor Views
- Bootstrap

## Architecture

- Repository Pattern
- Unit of Work Pattern
- SOLID Principles
